
# MAZE-GAME

A dynamic, randomly generated 3D VR maze survival game where no two playthroughs are the same. Players must navigate complex procedural routes, manage their health, and survive environmental hazards to conquer each level.

## Core Features
- **Procedural Maze Generation:** Each level is dynamically generated, requiring players to learn new routes and adapt to a completely unique maze layout upon replay.
- **Environmental Hazards:** Jump over dynamic Lava Blocks to survive; mistiming jumps results in health penalties.
- **Health Management:** Features a persistent health bar system that tracks player vitality across the maze, adding a layer of resource management to the puzzle-solving.

## Controls
- **W:** Jump Up
- **A:** Move Left
- **S:** Lower down
- **D:** Move Right
